Classes

A CSS Class is an identifier of an element which can be used multiple times on a page.

Memory device: Classes in school are periods. Remember to us a . whenever you want to reference a class.

  • Classes start with . in the selector .anything-you-want-to-name-it
  • Again, the same class name can be used more than once within the same file
  • The class keyword refers to the class name that you set in the html page in the specific tag.
  • In the CSS file, you can identify a class on the whole, or you can identify particluar parts to style (ex: p tags, h3, or li tags)
